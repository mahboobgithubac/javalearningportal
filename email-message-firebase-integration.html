📧 Email Alerts – Free or Chargeable?

Yes, email alerts can be free, depending on how many emails you send and the provider you use.

💰 Email Service Provider Comparison

Provider Free Tier Daily Limit Notes
Gmail (personal) ✅ Free ~100–150/day Best for personal/testing; not scalable
SendGrid ✅ 100/day ~3,000/month Requires API key & domain setup
Mailgun ✅ 5,000 first month Then paid Developer-friendly with APIs
Amazon SES ✅ 62,000/month (from EC2) Scalable Best for production-scale alerts

📦 Firebase + Gmail SMTP Example

You can use Gmail SMTP in Firebase Cloud Functions like this:

const nodemailer = require("nodemailer");

exports.sendEmailAlert = functions.firestore
  .document("orders/{orderId}")
  .onCreate(async (snap, context) => {
    const order = snap.data();

    let transporter = nodemailer.createTransport({
      service: 'gmail',
      auth: {
        user: 'your_email@gmail.com',
        pass: 'your_app_password' // Use App Password
      }
    });

    let info = await transporter.sendMail({
      from: '"Order Alerts" <your_email@gmail.com>',
      to: 'destination_email@gmail.com',
      subject: "🛒 New Order Received",
      text: `Order: ${order.product} x ${order.quantity}`
    });

    console.log("Email sent: ", info.messageId);
  });
🔐 Security Tip: Use App Passwords if 2FA is enabled on Gmail.
