üîç What is a Method Reference?

A method reference in Java is a shorthand for a lambda expression that calls a method. It uses the :: operator.

üìå Syntax:

ClassName::methodName

It can be used only when the lambda method body calls a method with the same arguments.

‚úÖ Types of Method References

Type Syntax Description Example
Static method ClassName::staticMethod Refers to a static method Math::abs
Instance method of a particular object object::instanceMethod Refers to a method of a specific object System.out::println
Instance method of an arbitrary object of a type ClassName::instanceMethod Method of any instance of the class String::toLowerCase
Constructor reference ClassName::new Refers to a class constructor ArrayList::new

‚úÖ Examples

1. Static Method Reference

public class StaticExample {
    public static int square(int n) {
        return n * n;
    }

    public static void main(String[] args) {
        Function<Integer, Integer> func = StaticExample::square;
        System.out.println(func.apply(5)); // Output: 25
    }
}

2. Instance Method of a Particular Object

Consumer<String> printer = System.out::println;
printer.accept("Hello from method reference!");

3. Instance Method of an Arbitrary Object

List<String> list = Arrays.asList("Java", "Python", "Go");
list.forEach(System.out::println);

4. Constructor Reference

Supplier<ArrayList<String>> supplier = ArrayList::new;
ArrayList<String> list = supplier.get();
list.add("Hello");
System.out.println(list);

üß† When to Use Method References?

  • When the lambda is just a method call
  • To make code cleaner and easier to read

‚úÖ Summary

  • Method references are a shorthand for simple method-calling lambdas.
  • Use :: to refer to static, instance, or constructor methods.
  • They improve readability and reduce verbosity.
