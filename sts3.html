üöÄ How to Push Code to GitHub from STS

  1. Right-click your project > Team > Share Project
  2. Select Git and click Next
  3. Create or select a repository
  4. Go to Git Staging view (Window > Show View > Other > Git > Git Staging)
  5. Add files to staging area, write commit message and Commit
  6. Click on the repo > Push Branch
  7. Add your GitHub remote URL and credentials
  8. Push to GitHub!

‚úÖ How to Add Lombok to STS

  1. Add Lombok to pom.xml: 
    <dependency>
  <groupId>org.projectlombok</groupId>
  <artifactId>lombok</artifactId>
  <version>1.18.32</version>
  <scope>provided</scope>
</dependency>
  2. Download Lombok jar from official site
  3. Run with java -jar lombok.jar
  4. Choose STS IDE path (e.g., SpringToolSuite4.ini)
  5. Restart STS
  6. Use Lombok annotations like @Getter, @Setter, etc.
