Java Interview Questions

1. What are the different ways to create an object in Java?

There are 5 main ways to create an object in Java:

  1. Using new keyword
    MyClass obj = new MyClass();
  2. Using Class.forName() (Dynamic class loading)
    Class<?> clazz = Class.forName("com.example.MyClass");
    Object obj = clazz.newInstance(); // Deprecated in Java 9+
  3. Using clone() method
    MyClass obj1 = new MyClass();
    MyClass obj2 = (MyClass) obj1.clone();
  4. Using ObjectInputStream (Deserialization)
    ObjectInputStream in = new ObjectInputStream(new FileInputStream("file.ser"));
    MyClass obj = (MyClass) in.readObject();
  5. Using Reflection with Constructor
    Constructor<MyClass> constructor = MyClass.class.getConstructor();
    MyClass obj = constructor.newInstance();

2. What is the purpose of Constructor class in Java?

The Constructor class is part of the java.lang.reflect package.

Purpose:

  • Allows reflective access to class constructors at runtime.
  • Supports dynamic object creation even for private/protected constructors.
  • Used in frameworks like Spring and Hibernate for dependency injection.

Example:

Constructor<MyClass> constructor = MyClass.class.getConstructor(String.class);
MyClass obj = constructor.newInstance("argument");

3. Differences Between Interface and Abstract Class After Java 8

Feature Interface Abstract Class
Inheritance Multiple Single
Method types abstract, default, static (Java 8+), private (Java 9+) abstract, concrete
Variables public static final only Any type allowed
Constructors Not allowed Can have constructors
Access Modifiers Only public public, protected, private
Use Case For capability (what class can do) For base behavior (what class is)

Java 8 Enhancements to Interface:

  • Default methods
  • Static methods
  • Private methods (Java 9+)

4. What Is a Concrete Class?

A Concrete class in Java is a class that:

  • Is not abstract
  • Can be instantiated directly
  • Implements all abstract methods (if any)

Example:

public class Dog extends Animal {
  @Override
  void makeSound() {
    System.out.println("Bark");
  }
}
