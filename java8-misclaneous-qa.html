Answers

Q1: Convert a list of strings to uppercase
names.stream().map(String::toUpperCase).collect(Collectors.toList())
Q2: Square a list of numbers
nums.stream().map(n -> n * n).collect(Collectors.toList())
Q3: Flatten a list of lists
nested.stream().flatMap(List::stream).collect(Collectors.toList())
Q4: Remove null or empty strings
mixed.stream().filter(s -> s != null && !s.isEmpty()).collect(Collectors.toList())
Q5: Convert a list to CSV string
nums.stream().map(String::valueOf).collect(Collectors.joining(", "))
Q40: Replace vowels with *
"education".chars().mapToObj(c -> "aeiou".indexOf(c) >= 0 ? "*" : "" + (char) c).collect(Collectors.joining())
