🚀 Add New Files and Deploy with GitHub + Netlify

Follow these steps to add new HTML, CSS, or JS files to your project and deploy them using GitHub and Netlify.

📁 1. Place New Files

Add your new files (e.g., spring-boot-interview.html, new-style.css) into your main/ folder or project root.

🧾 2. Stage the New Files

From your project folder, open terminal or command prompt and run:

git add .

This adds all new or modified files. To add only HTML files:

git add *.html

💬 3. Commit the Changes

Write a meaningful commit message:

git commit -m "Add new tutorial page: spring-boot-interview.html"

🚀 4. Push to GitHub

Push your changes to the main branch (or master if that’s your branch name):

git push origin main

🌐 5. Netlify Auto Deploys

Netlify will automatically detect the GitHub changes and deploy your new pages in about 30 seconds.

✅ 6. Access the New Page

Visit your live page at:

https://your-site-name.netlify.app/spring-boot-interview.html
