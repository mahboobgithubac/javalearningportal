Java Features from Java 9 to Java 17

A comprehensive guide to new features added post Java 8

Java 9 (Sep 2017)

Highlights
  • Jigsaw Project (Modules System)
  • JShell (REPL)
  • Improved Javadoc with Search
  • Private Interface Methods
  • Collection Factory Methods
  • Stream API Enhancements: takeWhile, dropWhile, etc.
  • Try-With-Resources Enhancement
  • Compact Strings for memory optimization

Java 10 (Mar 2018)

Highlights
  • var Keyword for local variable inference
  • Application Class-Data Sharing (AppCDS)
  • Garbage Collector Interface
  • Parallel Full GC for G1
  • Default Root Certificates

Java 11 (Sep 2018) - LTS

Highlights
  • var in Lambda Parameters
  • New String Methods: isBlank, lines, strip, repeat
  • New File Methods: readString, writeString
  • Standardized HttpClient
  • Launch Single-File Java Programs
  • Removed Java EE and CORBA modules

Java 12 to 17 Highlights

Java 12 (Mar 2019)
  • Switch Expressions (Preview)
  • Compact Number Formatting
  • Shenandoah GC (Experimental)
Java 13 (Sep 2019)
  • Text Blocks (Preview)
  • Switch Expressions (2nd Preview)
Java 14 (Mar 2020)
  • Switch Expressions (Standard)
  • Helpful NullPointerException Messages
  • Records (Preview)
  • Pattern Matching for instanceof (Preview)
Java 15 (Sep 2020)
  • Sealed Classes (Preview)
  • Text Blocks (Standard)
  • Hidden Classes
Java 16 (Mar 2021)
  • Records (Standard)
  • Pattern Matching for instanceof (Standard)
  • Sealed Classes (2nd Preview)
  • Vector API (Incubator)
Java 17 (Sep 2021) - LTS
  • Sealed Classes (Standard)
  • Pattern Matching for switch (Preview)
  • Enhanced Random Generators
  • Foreign Function & Memory API (Incubator)
  • Removed RMI, Applet, Security Manager
