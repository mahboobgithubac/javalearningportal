Find Second Occurrence of a Character (Java)

public class SecondOccurrenceFinder {
    public static void main(String[] args) {
        String input = "hello world";
        char target = 'l';

        int firstIndex = input.indexOf(target);
        if (firstIndex != -1) {
            int secondIndex = input.indexOf(target, firstIndex + 1);
            if (secondIndex != -1) {
                System.out.println("Second occurrence of '" + target + "' is at index: " + secondIndex);
            } else {
                System.out.println("Character occurs only once.");
            }
        } else {
            System.out.println("Character not found.");
        }
    }
}

  • indexOf(target) returns the index of the first occurrence.
  • indexOf(target, firstIndex + 1) starts the search from the next character after the first match.
  • If the second index is -1, the character appears only once.

Find Second Occurrence of Character using Java 8 Stream API

import java.util.stream.IntStream;

public class SecondOccurrenceWithStreams {
    public static void main(String[] args) {
        String input = "hello world";
        char target = 'l';

        int[] indexes = IntStream.range(0, input.length())
                .filter(i -> input.charAt(i) == target)
                .limit(2) // limit to first two occurrences
                .toArray();

        if (indexes.length < 2) {
            System.out.println("Second occurrence not found.");
        } else {
            System.out.println("Second occurrence of '" + target + "' is at index: " + indexes[1]);
        }
    }
}
üîç Explanation:
  • IntStream.range(0, input.length()) generates a stream of indices from 0 to the end of the string.
  • filter(i -> input.charAt(i) == target) filters the indices where the character matches the target.
  • limit(2) ensures only the first two matching indices are collected.
  • toArray() converts the filtered stream into an array of matching indices.
  • If the array has fewer than 2 elements, the second occurrence doesn‚Äôt exist.
