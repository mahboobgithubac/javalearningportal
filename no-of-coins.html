Minimum Coins Calculator in Java

Greedy Algorithm Approach

Program Description

This Java program finds the minimum number of coins needed to make a given amount using a greedy algorithm. It tries to use the largest denominations first. The coin denominations are {10, 5, 2, 1}, and the program works efficiently for most practical scenarios.

Java Code

import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        int[] coins = {10, 5, 2, 1}; // sorted in descending order
        int amount = 108;
        int count = 0;

        for (int coin : coins) {
            if (amount >= coin) {
                int used = amount / coin;
                count += used;
                amount = amount % coin;
            }
        }

        System.out.println("Minimum number of coins: " + count);
    }
}

Sample Output

Minimum number of coins: 13
