GEMs Tracking Tool

Architecture Overview

1. Architecture Type

Full Stack Web Application using Microservices-inspired layered architecture.

Technology Stack:
  • Frontend: ReactJS + Bootstrap (Client-Side Rendering)
  • Backend: Spring Boot (REST APIs)
  • Database: MySQL
  • Communication Format: JSON over HTTP
  • Build & Dependency: Maven
  • Server: Apache Tomcat (Embedded via Spring Boot)

ðŸ”¹ Database Layer (MySQL)

4. Security and Validation

  • Input validations on both frontend (React) and backend (Spring Boot)
  • Role-based access control (optional extension)
  • Field-level validation using JSR-303 (Hibernate Validator)

5. Tools and DevOps

  • Build Tool: Maven
  • Server: Embedded Tomcat via Spring Boot
  • Version Control: Git (if used)
  • Deployment: WAR/JAR file deployed to server or Docker container (optional)
  • Testing: JUnit, Postman for API testing
