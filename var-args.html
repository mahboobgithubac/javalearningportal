‚úÖ Varargs (Variable Arguments) in Java

Varargs (variable-length arguments) allow a method to accept zero or more arguments of a specified type. It's a cleaner alternative to using arrays or method overloading when the number of arguments is unknown.

üß† Syntax

returnType methodName(type... varName)
  • type... indicates a varargs parameter.
  • Internally, Java treats varargs as an array.
‚úÖ Only one varargs is allowed per method, and it must be the last parameter.

üìò Example: Sum of Integers using Varargs

public class VarargsExample {

    public static int sum(int... numbers) {
        int total = 0;
        for (int n : numbers) {
            total += n;
        }
        return total;
    }

    public static void main(String[] args) {
        System.out.println(sum());              // Output: 0
        System.out.println(sum(10));            // Output: 10
        System.out.println(sum(1, 2, 3, 4, 5));  // Output: 15
    }
}
