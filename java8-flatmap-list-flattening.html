Java 8 flatMap() Example

‚úÖ Scenario: Flatten a List of Lists

Use flatMap() to convert a nested structure like List<List<String>> into a single flat List<String>.

üßæ Java Code:

import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class FlatMapExample {
  public static void main(String[] args) {
    List<List<String>> namesNested = Arrays.asList(
      Arrays.asList("John", "Jane"),
      Arrays.asList("Tom", "Jerry"),
      Arrays.asList("Alice", "Bob")
    );

    // Flattening the nested list using flatMap
    List<String> flatList = namesNested.stream()
      .flatMap(List::stream)
      .collect(Collectors.toList());

    System.out.println(flatList);
  }
}
Output:
[John, Jane, Tom, Jerry, Alice, Bob]

üîç How flatMap() Works:

  • namesNested.stream() ‚Üí Creates a stream of lists: Stream<List<String>>
  • .flatMap(List::stream) ‚Üí Converts each sublist to a stream and flattens into one stream: Stream<String>
  • .collect(Collectors.toList()) ‚Üí Collects all elements into one flat List<String>
