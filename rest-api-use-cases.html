REST API Use Cases with Custom Logic

1. Employee Leave Management System
  • Custom Logic:
    • Validate leave balance before approval
    • Reject overlapping requests
    • Send email notifications
  • Endpoints:
    • POST /leave/apply
    • GET /leave/status/{employeeId}
    • POST /leave/approve/{requestId}
2. Expense Reimbursement System
  • Custom Logic:
    • Validate receipt and submission window
    • Auto-classify expense types
  • Endpoints:
    • POST /expenses/submit
    • GET /expenses/employee/{id}
    • POST /expenses/approve/{expenseId}
3. Loan Eligibility Checker
  • Custom Logic:
    • Check eligibility based on age, salary, credit score
    • Calculate eligible amount using a formula
  • Endpoints:
    • POST /loan/check-eligibility
    • GET /loan/offers/{customerId}
4. Dynamic Pricing API
  • Custom Logic:
    • Price changes with demand, time, or location
    • Handle discounts and surge pricing
  • Endpoints:
    • GET /price/product/{productId}?location=&time=
    • POST /price/apply-discount
5. Study Tracker API
  • Custom Logic:
    • Auto-calculate completion progress
    • Auto-archive completed tasks
    • Task urgency based on due date
  • Endpoints:
    • POST /study/add-task
    • GET /study/tasks?status=pending
    • GET /study/progress-summary
6. Custom URL Shortener
  • Custom Logic:
    • Generate base62 encoded short links
    • Expire after time/click limit
    • Track click analytics
  • Endpoints:
    • POST /shorten
    • GET /{shortUrl}
    • GET /analytics/{shortUrl}
7. Inventory Auto-Replenishment
  • Custom Logic:
    • Trigger reorder when below threshold
    • Use demand prediction
    • Send restock alerts
  • Endpoints:
    • GET /inventory/{itemId}
    • POST /inventory/update-stock
    • GET /inventory/restock-suggestions
8. Interview Question Tracker
  • Custom Logic:
    • Assign difficulty based on tags and performance
    • Personalize questions for each user
    • Track time spent per question
  • Endpoints:
    • POST /questions/add
    • GET /questions/personalized/{userId}
    • GET /questions/summary
9. PDF Report Generator
  • Custom Logic:
    • Generate reports dynamically from DB
    • Add charts, watermark, branding
    • Allow download or email
  • Endpoints:
    • GET /report/generate?type=monthly
    • GET /report/download/{reportId}
10. Authentication with OTP
  • Custom Logic:
    • Generate and validate 6-digit OTP
    • Limit retry attempts, expire after 5 mins
    • Lock account on multiple failures
  • Endpoints:
    • POST /auth/send-otp
    • POST /auth/verify-otp
    • POST /auth/login
