Java 8 Array Problem Solving Cheat Sheet

Given Integer array: 
int[] numbers = {1, 2, 3, 2, 4, 1, 5, 6, 5};

 int[] arr = {1, 2, 3, 2, 4, 1, 5, 6, 5, 12, 15, 42};
int[] arr2 = {7, 8, 9};

1. Find the Maximum Number

int max = Arrays.stream(arr).max().orElseThrow();

Output: Max = 6

2. Find the Minimum Number

int min = Arrays.stream(arr).min().orElseThrow();

Output: Min = 1

3. Calculate the Sum

int sum = Arrays.stream(arr).sum();

Output: Sum = 16

4. Calculate the Average

double average = Arrays.stream(arr).average().orElse(0.0);

Output: Average = 3.2

5. Count Even Numbers

long evenCount = Arrays.stream(arr).filter(n -> n % 2 == 0).count();

Output: Even Count = 2

6. Get a List of Squares

List<Integer> squares = Arrays.stream(arr)\n    .map(n -> n * n)\n    .boxed()\n    .collect(Collectors.toList());

Output: Squares = [1, 4, 9, 16]

7. Filter Numbers Greater Than 10

List<Integer> result = Arrays.stream(arr)\n    .filter(n -> n > 10)\n    .boxed()\n    .collect(Collectors.toList());

Output: Result = [12, 15]

8. Remove Duplicates

int[] distinct = Arrays.stream(arr).distinct().toArray();

Output: Distinct = [1, 2, 3, 4, 5]

9. Sort in Ascending Order

int[] ascSorted = Arrays.stream(arr).sorted().toArray();

Output: Ascending = [1, 2, 3, 4, 5]

10. Sort in Descending Order

int[] descSorted = Arrays.stream(arr)\n    .boxed()\n    .sorted(Comparator.reverseOrder())\n    .mapToInt(Integer::intValue)\n    .toArray();

Output: Descending = [5, 4, 3, 2, 1]

11. Find Second Largest Number

OptionalInt secondLargest = Arrays.stream(arr)\n    .distinct()\n    .boxed()\n    .sorted(Comparator.reverseOrder())\n    .skip(1)\n    .findFirst()\n    .mapToInt(Integer::intValue);

Output: Second Largest = 4

12. Check if Array Contains 42

boolean contains = Arrays.stream(arr).anyMatch(n -> n == 42);

Output: Contains = true/false

13. Convert int[] to List

List<Integer> list = Arrays.stream(arr).boxed().collect(Collectors.toList());

Output: List = [1, 2, 3]

14. Merge Two Arrays

int[] merged = IntStream.concat(Arrays.stream(arr1), Arrays.stream(arr2)).toArray();

15. Find Duplicates in Integer Array

Set<Integer> seen = new HashSet<>();\nSet<Integer> duplicates = Arrays.stream(arr)\n    .boxed()\n    .filter(n -> !seen.add(n))\n    .collect(Collectors.toSet());

Output: Duplicates = [1, 2, 5]
