Git Integration (Push to GitHub from STS)

  1. Right-click your project → Team → Share Project
  2. Select Git and click Next
  3. Create/select a Git repository location
  4. Go to Git Staging view (Window → Show View → Other → Git → Git Staging)
  5. Add files to the staging area, write a commit message, and click Commit
  6. Right-click the project → Team → Push Branch
  7. Enter your GitHub repository URL and credentials
  8. Click Next and then Finish to push your code

How to Add Lombok to Spring Tool Suite (STS)

  1. Add the Lombok dependency to your pom.xml: 
    <dependency>
  <groupId>org.projectlombok</groupId>
  <artifactId>lombok</artifactId>
  <version>1.18.32</version>
  <scope>provided</scope>
</dependency>
  2. Download the Lombok jar from projectlombok.org
  3. Run it using: java -jar lombok.jar
  4. Select your STS installation directory
  5. Click Install/Update and restart STS
  6. Now you can use annotations like @Getter, @Setter, @Builder, etc.

Keyboard Shortcuts in Spring Tool Suite (STS)

  • Ctrl + Shift + R – Open Resource
  • Ctrl + Shift + T – Open Type (Class/Interface)
  • Alt + Shift + R – Rename
  • Ctrl + O – Show Outline of Current Class
  • Ctrl + Shift + F – Format Source Code
  • F3 – Go to Declaration
  • Ctrl + 1 – Quick Fix / Suggestion
