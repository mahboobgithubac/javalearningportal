Java 9 to 17 - Short Descriptions

  • Java 9 (2017): Introduced the Java Platform Module System (JPMS) for modularizing code, JShell (an interactive REPL), and improved APIs and performance.
  • Java 10 (2018): Added var keyword for local variable type inference, along with minor performance and garbage collection enhancements.
  • Java 11 (2018 - LTS): A Long-Term Support version that added HTTP Client API, removed deprecated modules (like JavaFX), and improved performance and security.
  • Java 12 (2019): Introduced Switch Expressions (preview) for cleaner switch statements and performance improvements.
  • Java 13 (2019): Brought Text Blocks (preview) for multiline strings and more enhancements in memory and performance.
  • Java 14 (2020): Finalized Switch Expressions, added Helpful NullPointerExceptions, and introduced Records and Pattern Matching (preview).
  • Java 15 (2020): Added Sealed Classes (preview), finalized Text Blocks, and included more garbage collector updates.
  • Java 16 (2021): Finalized Records, added Pattern Matching for instanceof, and improved performance with new JEPs.
  • Java 17 (2021 - LTS): Another Long-Term Support release featuring finalized Sealed Classes, enhanced performance, new language features, and long-term stability for enterprise use.
