Java Method Reference Explained

✅ What is a Method Reference?

A method reference in Java 8 is a shorthand notation of a lambda expression that calls an existing method. It uses the :: operator and improves code readability by eliminating boilerplate lambda syntax.

✅ Types of Method References

Type Syntax Example
Static Method ClassName::staticMethod Math::abs
Instance Method of Particular Object object::instanceMethod scanner::nextLine
Instance Method of Arbitrary Object ClassName::instanceMethod String::toLowerCase
Constructor Reference ClassName::new ArrayList::new

✅ Example 1: Static Method Reference

public class Utility {
    public static void greet(String name) {
        System.out.println("Hello, " + name);
    }
}

Consumer<String> greeter = Utility::greet;
greeter.accept("Alice"); // Output: Hello, Alice

✅ Example 2: Instance Method Reference

public class Printer {
    public void print(String msg) {
        System.out.println("Message: " + msg);
    }
}

Printer printer = new Printer();
Consumer<String> printRef = printer::print;
printRef.accept("Hello world");

✅ Example 3: Method Reference with Arguments

public class MathUtil {
    public static int add(int a, int b) {
        return a + b;
    }
}

// Method reference
BiFunction<Integer, Integer, Integer> adder = MathUtil::add;

// Pass arguments
int result = adder.apply(10, 5);  // Output: 15

✅ Example 4: Method Reference in Streams

public class MyUtil {
    public static String shout(String word) {
        return word.toUpperCase() + "!";
    }
}

List<String> words = Arrays.asList("hi", "java", "code");

List<String> shouted = words.stream()
    .map(MyUtil::shout)
    .collect(Collectors.toList());

✅ Example 5: Lambda to Method Reference

// Lambda
BiFunction<Integer, Integer, Integer> add = (a, b) -> a + b;

// Equivalent Method Reference
BiFunction<Integer, Integer, Integer> add = MathUtil::add;

✅ Summary Table

Lambda Expression Method Reference Equivalent
(a, b) -> a + b MathUtil::add
s -> s.toUpperCase() String::toUpperCase
x -> System.out.println(x) System.out::println

✅ Final Interview Line

"A method reference is a cleaner alternative to a lambda when you're only calling an existing method. It's useful in stream operations, functional interfaces, and simplifies code. The method reference syntax ClassName::methodName directly maps to the method signature expected by the functional interface."
