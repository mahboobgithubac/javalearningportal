Java Developer Interview Guide (6 Years Experience)

1. ‚úÖ Core Java (Java 8+ Mandatory)

Deep understanding is expected.

Topics to master:
  • OOP Principles
  • Exception handling
  • Collections (List, Map, Set, Queue)
  • Concurrency: Thread, ExecutorService, synchronized, Locks
  • Java 8 Features: Streams, Lambda, Optional, Functional interfaces
  • equals(), hashCode(), Comparable vs Comparator
  • Memory management, Garbage collection
Sample Questions:
  • How does HashMap work internally?
  • Explain Stream.map() vs Stream.flatMap().
  • What are the differences between synchronized and Lock?

2. ‚úÖ Spring Framework & Spring Boot

At 6 years, you must be fluent in the Spring ecosystem.

Topics to focus on:
  • Spring Core: DI, Beans, Scopes, Lifecycle
  • Spring Boot: Autoconfiguration, starters
  • Spring MVC: REST controllers
  • Spring Data JPA: Repository pattern, custom queries
  • Spring Security: Filters, JWT, authentication
  • Spring AOP: Cross-cutting concerns
Sample Questions:
  • How does Spring Boot autoconfiguration work?
  • How to secure a REST API using JWT?
  • Difference between @Component, @Service, and @Repository?

3. ‚úÖ RESTful API Design

Focus areas:
  • HTTP methods, status codes
  • URI design best practices
  • Exception handling (@ControllerAdvice)
  • Versioning, Pagination
  • Postman testing
Sample Questions:
  • What is idempotency? How do you ensure it in APIs?
  • How to handle error responses in REST?
  • How to version your REST APIs?

4. ‚úÖ Database (SQL & JPA/Hibernate)

Topics:
  • Joins, Subqueries, Indexing
  • Transactions, ACID properties
  • Hibernate caching, lazy/eager loading
  • Named/native queries
  • Criteria API
Sample Questions:
  • Difference between INNER JOIN and LEFT JOIN?
  • How does Hibernate manage relationships?
  • Explain N+1 problem in Hibernate.

5. ‚úÖ Microservices Architecture

Topics:
  • REST vs Microservices
  • Service communication (REST, Feign, Kafka)
  • Spring Cloud: Config, Eureka, Gateway
  • Circuit breaker (Resilience4j/Hystrix)
  • Centralized logging and config
Sample Questions:
  • How do microservices communicate securely?
  • Explain API Gateway vs Service Discovery.
  • How do you handle service failure in microservices?

6. ‚úÖ DevOps & CI/CD (Optional but Valued)

Topics:
  • Jenkins, GitHub Actions
  • Docker basics: image, container
  • Kubernetes concepts
  • Application monitoring (Prometheus, ELK)
Sample Questions:
  • How do you containerize a Spring Boot app?
  • What are the stages in your CI/CD pipeline?

7. ‚úÖ Testing

Tools:
  • JUnit 5, Mockito
  • Spring Boot Test
  • Testcontainers (optional)
  • Postman/TestNG (optional)
Sample Questions:
  • How do you mock a service in Spring Boot?
  • What is the difference between unit and integration testing?

  • How to scale an order processing system?
