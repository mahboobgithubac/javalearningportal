‚úÖ Passing Data to Abstract Class Constructor in Java

üìå Overview

  • Abstract classes in Java can have constructors.
  • These constructors are used to initialize common fields.
  • To pass data, the subclass must use super(arguments) in its constructor.

üî∏ Example Code

abstract class Person {
    String name;

    // Abstract class constructor
    public Person(String name) {
        this.name = name;
        System.out.println("Person constructor called with: " + name);
    }

    abstract void display();
}

class Student extends Person {
    int rollNo;

    // Subclass constructor calling abstract class constructor
    public Student(String name, int rollNo) {
        super(name);  // Passing data to abstract class
        this.rollNo = rollNo;
    }

    @Override
    void display() {
        System.out.println("Name: " + name + ", Roll No: " + rollNo);
    }
}

public class Main {
    public static void main(String[] args) {
        Student s = new Student("Alice", 101);
        s.display();
    }
}

üßæ Output

Person constructor called with: Alice
Name: Alice, Roll No: 101

üß† Explanation

  • Person is an abstract class with a constructor that initializes the name.
  • Student extends Person and uses super(name) to call the parent constructor.
  • The abstract class constructor is executed before the subclass constructor finishes.
