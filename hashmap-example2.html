HashMap Overview: A HashMap is a key-value based data structure from the Java Collections Framework that allows fast retrieval, insertion, and deletion. Keys must be unique and are indexed by their hash codes.

1. Creating and Iterating a HashMap

Map<String, Integer> map = new HashMap<>();
map.put("A", 1);
map.put("B", 2);
map.put("C", 3);

1.1 Using entrySet()

for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " => " + entry.getValue());
}
Output: 
A => 1
B => 2
C => 3

1.2 Using keySet()

for (String key : map.keySet()) {
    System.out.println(key + " => " + map.get(key));
}

1.3 Using forEach()

map.forEach((key, value) -> {
    System.out.println(key + " => " + value);
});

1.4 Using Iterator

Iterator<Map.Entry<String, Integer>> iterator = map.entrySet().iterator();
while (iterator.hasNext()) {
    Map.Entry<String, Integer> entry = iterator.next();
    System.out.println(entry.getKey() + " => " + entry.getValue());
}

1.5 Using Stream API

map.entrySet().stream()
   .forEach(entry -> System.out.println(entry.getKey() + " => " + entry.getValue()));

2. Updating HashMap Values

2.1 Using put()

map.put("A", 10); // Overwrites if "A" exists

2.2 Using replace()

map.replace("B", 20); // Replaces if "B" exists
map.replace("C", 3, 30); // Only replaces if current value is 3

2.3 Using compute() and computeIfPresent()

map.compute("A", (k, v) -> v + 5);
map.computeIfPresent("B", (k, v) -> v * 2);

2.4 Using merge()


 Map map = new HashMap<>();
map.put("C", 3);

map.merge("C", 5, Integer::sum);  //add 5 and 3 
System.out.println(map.get("C"));

3. Updating Values While Iterating

for (Map.Entry<String, Integer> entry : map.entrySet()) {
    entry.setValue(entry.getValue() + 1);
}

Alternative methods:

map.forEach((k, v) -> map.replace(k, v + 1));
map.keySet().forEach(k -> map.compute(k, (key, val) -> val + 1));

4. Can You Update a Key in HashMap?

No, keys cannot be directly updated. The best approach is to remove the old key and insert a new one:

if (map.containsKey("A")) {
    Integer value = map.remove("A");
    map.put("Alpha", value);
}

Never modify fields in custom objects that are used in equals() or hashCode(). Doing so may corrupt the internal structure of the HashMap.

class Student {
    int id;
    // equals and hashCode use id
}
Student s = new Student(101);
map.put(s, "Alice");
s.id = 102;
System.out.println(map.get(s)); // Likely null

5. Summary

  • Use entrySet() for safe iteration and updating.
  • Use put/replace/compute/merge to update values.
  • To change a key, remove old and insert new.
  • Prefer immutable keys in real-world HashMap usage.
