Best Time to Buy and Sell Stock

Problem: Given an array of stock prices, find the maximum profit you can make by buying on one day and selling on another day in the future.

üîÅ Brute Force (O(n¬≤)) Approach

int[] arr = { 7, 1, 5, 3, 6, 4 };
int maxProfit = 0;

for (int i = 0; i < arr.length - 1; i++) {
    for (int j = i + 1; j < arr.length; j++) {
        int profit = arr[j] - arr[i];  // Buy at i, Sell at j
        if (profit > maxProfit)
            maxProfit = profit;
    }
}

System.out.println("maxProfit = " + maxProfit);

‚úÖ Optimized (O(n)) Approach

int[] prices = { 7, 1, 5, 3, 6, 4 };
int minPrice = Integer.MAX_VALUE;
int maxProfit = 0;

for (int price : prices) {
    if (price < minPrice) {
        minPrice = price;  // Update min price
    } else if (price - minPrice > maxProfit) {
        maxProfit = price - minPrice;  // Update profit
    }
}

System.out.println("Max Profit = " + maxProfit);

Note: The optimized solution is recommended for large input sizes (up to 105) due to its linear time complexity.
