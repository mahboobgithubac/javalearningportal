Java 8 - Different Ways to Iterate Through Arrays

Here are several efficient and modern ways to iterate through arrays in Java 8:

1. Using the Enhanced For Loop

This is the most straightforward and commonly used method:

int[] numbers = {1, 2, 3, 4, 5};
for (int number : numbers) {
    System.out.println(number);
}

2. Using Arrays.stream() and forEach()

This method converts the array into a stream for functional-style operations:

import java.util.Arrays;

int[] numbers = {1, 2, 3, 4, 5};
Arrays.stream(numbers).forEach(number -> System.out.println(number));

// Or using method reference
Arrays.stream(numbers).forEach(System.out::println);

3. Using IntStream.range() and forEach()

Useful when you need access to the index during iteration:

import java.util.stream.IntStream;

int[] numbers = {1, 2, 3, 4, 5};
IntStream.range(0, numbers.length)
         .forEach(i -> System.out.println("Index: " + i + ", Value: " + numbers[i]));

4. Using Arrays.stream() and forEachOrdered()

Preserves the order even when used with a parallel stream:

import java.util.Arrays;

Integer[] numbers = {1, 2, 3, 4, 5};
Arrays.stream(numbers)
      .parallel()
      .forEachOrdered(System.out::println);

Notes:

  • forEach() does not guarantee order in parallel streams.
  • forEachOrdered() maintains order even in parallel processing.
  • Use Arrays.stream() or IntStream.range() for primitive arrays.
  • Use Arrays.stream() for object arrays like Integer[].
