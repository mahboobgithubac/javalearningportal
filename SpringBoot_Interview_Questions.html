1. What is Spring Boot?
Spring Boot is a framework that simplifies the development of Spring applications by providing:

  • Auto-configuration
  • Embedded servers (like Tomcat)
  • Opinionated starter dependencies
  • Production-ready features (e.g., health checks, metrics)

2. What are the advantages of Spring Boot?

  • Reduces boilerplate code
  • No need for external server deployment
  • Auto-configuration based on classpath
  • Starter templates for dependencies
  • Production-ready features (Actuator)

3. What are Starters in Spring Boot?
Starters are pre-defined dependencies that simplify build configuration.

<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
</dependency>

4. How does Spring Boot auto-configuration work?
It uses @EnableAutoConfiguration and classpath scanning to automatically configure beans.

5. How do you create a Spring Boot application?

@SpringBootApplication
public class App {
  public static void main(String[] args) {
    SpringApplication.run(App.class, args);
  }
}

6. What is @SpringBootApplication?
A convenience annotation that includes:

  • @Configuration
  • @EnableAutoConfiguration
  • @ComponentScan

ðŸ”¹ Intermediate Level

7. What is Spring Boot Actuator?
Provides production-ready features like health checks, metrics, and environment info at endpoints like /actuator/health.

8. How do you handle exceptions in Spring Boot?

@ControllerAdvice
public class GlobalExceptionHandler {
  @ExceptionHandler(Exception.class)
  public ResponseEntity<String> handleException(Exception e) {
    return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).body(e.getMessage());
  }
}

11. What are Profiles in Spring Boot?
Used to define environment-specific configurations using @Profile.

12. How to configure properties in Spring Boot?
Use application.properties or application.yml and bind them using @Value or @ConfigurationProperties.

14. How do you test a Spring Boot application?
Use @SpringBootTest for integration tests and @WebMvcTest, @DataJpaTest for slice tests.

19. Difference between Spring Boot and Spring MVC?
Spring Boot simplifies configuration and includes embedded servers, while Spring MVC is the web framework part.

22. How do you implement security in Spring Boot?
Use Spring Security and configure it with SecurityFilterChain or WebSecurityConfigurerAdapter (deprecated in newer versions).

23. What is the difference between @RestController and @Controller?
@RestController = @Controller + @ResponseBody
Used for REST APIs that return JSON/XML directly.
