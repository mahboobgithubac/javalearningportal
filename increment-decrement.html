Java Operators: Increment, Decrement, Left Shift, Right Shift

üîº Increment and üîΩ Decrement Operators

These operators are used to increase or decrease a value by 1.

// Example:
public class IncDecExample {
    public static void main(String[] args) {
        int a = 5;

        System.out.println("a++ = " + (a++)); // prints 5, then a becomes 6
        System.out.println("++a = " + (++a)); // a is 7 now
        System.out.println("a-- = " + (a--)); // prints 7, then a becomes 6
        System.out.println("--a = " + (--a)); // a is 5 now
    }
}

Shifts bits to the left. Equivalent to multiplying by 2‚Åø.

// Example:
public class LeftShiftExample {
    public static void main(String[] args) {
        int x = 3; // 0011
        int result = x << 2; // 1100 = 12
        System.out.println("3 << 2 = " + result);
    }
}

Shifts bits to the right. Equivalent to dividing by 2‚Åø.

// Example:
public class RightShiftExample {
    public static void main(String[] args) {
        int x = 12; // 1100
        int result = x >> 2; // 0011 = 3
        System.out.println("12 >> 2 = " + result);
    }
}

Q1: What will be the output?
int x = 5;
System.out.println(++x + x++);
Answer: 12
Explanation: ++x is 6, x++ is 6 (then x becomes 7). So 6 + 6 = 12.
Q2: What is the result of this code?
int y = 8;
int result = y >> 1;
System.out.println(result);
Answer: 4
Q3: Predict the output:
int a = 2;
int b = a++ + ++a;
System.out.println(b);
Answer: 6
Explanation: a++ is 2, ++a is 4 (after a became 3), so 2+4 = 6.
