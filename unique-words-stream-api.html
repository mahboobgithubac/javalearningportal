Write a Java program that takes a sentence as input and prints all the unique words in ascending alphabetical order.

The program should:
  • Remove all punctuation from the sentence.
  • Consider words case-insensitively (e.g., "Java" and "java" are treated the same).
  • Extract all the words, eliminate duplicates, and sort them alphabetically.
  • Use Java 8 Stream API for processing.

Write a Java program using Java 8 Stream API to find all unique words from a given sentence and display them in ascending alphabetical order.

âœ… Java Code

import java.util.*;
import java.util.stream.*;

public class UniqueWordsSorted {
    public static void main(String[] args) {
        String input = "Java is fun. Java is powerful. Java is popular.";

        // Step 1: Normalize the string (remove punctuation, convert to lowercase)
        String cleaned = input.replaceAll("[^a-zA-Z ]", "").toLowerCase();

        // Step 2: Split into words and stream
        List<String> uniqueSortedWords = Arrays.stream(cleaned.split("\\s+"))
            .distinct()
            .sorted()
            .collect(Collectors.toList());

        // Output result
        System.out.println("Unique words in ascending order:");
        uniqueSortedWords.forEach(System.out::println);
    }
}

âœ… Explanation

  • replaceAll("[^a-zA-Z ]", ""): Removes punctuation.
  • toLowerCase(): Makes the comparison case-insensitive.
  • split("\\s+"): Splits the sentence into words.
  • distinct(): Removes duplicates.
  • sorted(): Sorts the words alphabetically.
  • collect(Collectors.toList()): Collects the results into a List.
