Java 8 Object List Problem Solving Questions & Solutions

1. Sort employees by name
employees.stream() .sorted(Comparator.comparing(e -> e.name)) .forEach(System.out::println);
2. Sort by salary descending
employees.stream() .sorted(Comparator.comparingDouble(e -> -e.salary)) .forEach(System.out::println);
3. Filter employees with salary > 50000
employees.stream() .filter(e -> e.salary > 50000) .forEach(System.out::println);
4. Find max salary employee
Employee maxSal = employees.stream() .max(Comparator.comparingDouble(e -> e.salary)) .orElse(null); System.out.println("Max salary: " + maxSal);
5. Find youngest employee
Employee youngest = employees.stream() .min(Comparator.comparingInt(e -> e.age)) .orElse(null); System.out.println("Youngest: " + youngest);
