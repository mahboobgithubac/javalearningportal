Java Inheritance and Polymorphism

Inheritance allows a subclass to acquire properties and behaviors of a superclass.

Polymorphism enables a single interface to represent different types of objects.

class Animal {
    void makeSound() {
        System.out.println("Some sound");
    }
}

class Dog extends Animal {
    void makeSound() {
        System.out.println("Bark");
    }
}

Here, Dog overrides makeSound demonstrating runtime polymorphism.
