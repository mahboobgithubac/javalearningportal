The Object class is the root class for all Java classes. It defines several useful methods available to all objects.

1. toString()

Returns a string representation of the object. Usually overridden to display custom info. 
class Student {
    String name = "Alice";
    int age = 22;

    public String toString() {
        return "Student{name='" + name + "', age=" + age + "}";
    }

    public static void main(String[] args) {
        Student s = new Student();
        System.out.println(s.toString());
    }
}
Output: Student{name='Alice', age=22}

2. equals(Object obj)

Compares two objects for equality. Can be overridden to define logical equality. 
class Person {
    String name;

    Person(String name) {
        this.name = name;
    }

    public boolean equals(Object obj) {
        Person p = (Person) obj;
        return this.name.equals(p.name);
    }

    public static void main(String[] args) {
        Person p1 = new Person("John");
        Person p2 = new Person("John");
        System.out.println(p1.equals(p2));
    }
}
Output: true

3. hashCode()

Returns an integer hash code value. Must be consistent with equals() if overridden. 
class Demo {
    int id;

    Demo(int id) {
        this.id = id;
    }

    public int hashCode() {
        return id * 31;
    }

    public static void main(String[] args) {
        Demo d = new Demo(7);
        System.out.println(d.hashCode());
    }
}
Output: 217

4. getClass()

Returns the runtime class of the object. 
public class Main {
    public static void main(String[] args) {
        String s = "Hello";
        System.out.println(s.getClass());
    }
}
Output: class java.lang.String

5. clone()

Creates and returns a copy (clone) of the object. The class must implement Cloneable. 
class Employee implements Cloneable {
    String name = "Tom";

    public Object clone() throws CloneNotSupportedException {
        return super.clone();
    }

    public static void main(String[] args) throws CloneNotSupportedException {
        Employee e1 = new Employee();
        Employee e2 = (Employee) e1.clone();
        System.out.println(e2.name);
    }
}
Output: Tom

6. finalize() (Deprecated)

Called by the garbage collector before object is destroyed. Deprecated since Java 9. 
class FinalizeDemo {
    protected void finalize() {
        System.out.println("Object is destroyed");
    }

    public static void main(String[] args) {
        FinalizeDemo obj = new FinalizeDemo();
        obj = null;
        System.gc();  // Suggests GC
    }
}
Output (may vary): Object is destroyed
