Deploy Spring Boot REST API on AWS Elastic Beanstalk

Follow these steps to deploy your Spring Boot application on AWS Elastic Beanstalk.

1. Prepare Your Spring Boot Application

  • Package the application using mvn clean package.
  • Ensure itâ€™s a runnable JAR file.
  • Set server.port=5000 in application.properties (Beanstalk expects port 5000).

2. Install the Elastic Beanstalk CLI (Optional)

pip install awsebcli --upgrade --user

3. Create a Beanstalk Application

  • Go to AWS Elastic Beanstalk Console.
  • Click "Create Application".
  • Choose "Java" as the platform.
  • Upload your packaged JAR file.

4. Deploy the Application

Elastic Beanstalk will automatically create EC2, load balancers, and other resources.

5. Access Your API

Open the Beanstalk-generated URL, e.g., http://yourapp-env.elasticbeanstalk.com.
