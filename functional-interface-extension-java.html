Yes! In Java, a functional interface can extend another interface if:

  • It doesn't add another abstract method, OR
  • It only inherits a single abstract method, possibly with the same signature

‚úÖ Example 1: Extending Another Functional Interface

@FunctionalInterface
interface A {
    void doSomething();
}

@FunctionalInterface
interface B extends A {
    // Still a functional interface
}

    void anotherThing(); // ‚ùå Two abstract methods now
}

‚úÖ Example 3: Extending a Non-Functional Interface

    default void info() {
        System.out.println("I'm a marker interface");
    }

@FunctionalInterface

    void run();
}

‚úÖ Example 4: Overlapping Abstract Methods (Same Signature)

@FunctionalInterface
interface X {
    void process(String s);
}

@FunctionalInterface
interface Y {
    void process(String s);
}

@FunctionalInterface
interface Z extends X, Y {
    // Valid since both signatures are the same
}
