Lambda Expressions in Java

✅ What is a Lambda Expression?

A lambda expression is an anonymous function in Java. Introduced in Java 8, it allows you to pass behavior as parameters.

✅ Syntax:

(parameter1, parameter2) -> { // body }

✅ Why use it?

  • Shorter, cleaner code
  • Ideal for functional interfaces
  • Powerful with streams and collections

✅ Example 1: Iterating with Lambda

import java.util.Arrays;
import java.util.List;

public class LambdaExample {
    public static void main(String[] args) {
        List<String> names = Arrays.asList("Mahboob", "Anwar", "John", "Alex");

        names.forEach(name -> System.out.println("Hello " + name));
    }
}

✅ Example 2: Functional Interface with Lambda

@FunctionalInterface
interface Greeting {
    void sayHello(String name);
}

public class LambdaGreeting {
    public static void main(String[] args) {
        Greeting greet = (name) -> System.out.println("Hi " + name);
        greet.sayHello("Mahboob");
    }
}

✅ Use Cases

  • Collections API: forEach, map, filter
  • Event Listeners
  • Runnable and Callable tasks
