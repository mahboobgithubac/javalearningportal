üîπ What is an Interface?

An interface in Java is a reference type, similar to a class, that can contain only constants, method signatures, default methods, static methods, and nested types. The methods in interfaces are implicitly public and abstract.

üî∏ Simple Interface Example

interface Animal {
    void sound(); // abstract method
}

class Dog implements Animal {
    public void sound() {
        System.out.println("Dog barks");
    }
}

class Cat implements Animal {
    public void sound() {
        System.out.println("Cat meows");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal dog = new Dog();
        Animal cat = new Cat();

        dog.sound();  // Output: Dog barks
        cat.sound();  // Output: Cat meows
    }
}

üß† Explanation

  • Animal is an interface with an abstract method sound().
  • Dog and Cat implement the Animal interface.
  • Each class provides its own implementation of the sound() method.

üß™ Interface with Variables

interface Constants {
    int MAX = 100; // implicitly public static final
    String APP_NAME = "TestApp";
}

Interface variables are by default public static final and can be accessed without an object.

üß© Interface Inheritance

interface A {
    void methodA();
}

interface B extends A {
    void methodB();
}

An interface can extend another interface using the extends keyword.

‚öôÔ∏è Multiple Interface Implementation

interface Printable {
    void print();
}

interface Showable {
    void show();
}

class Demo implements Printable, Showable {
    public void print() {
        System.out.println("Printing...");
    }

    public void show() {
        System.out.println("Showing...");
    }
}

Java allows a class to implement multiple interfaces, achieving multiple inheritance.
