✅ Java Ternary Operator

The ternary operator in Java is a shorthand for the if-else statement. It’s used to evaluate a boolean expression and return one of two values depending on the result.

📌 Syntax:

condition ? expression_if_true : expression_if_false;
The ternary operator is the only operator in Java that takes three operands.

🔍 Example 1: Finding Maximum of Two Numbers

public class TernaryExample {
    public static void main(String[] args) {
        int a = 10, b = 20;

        // Using ternary operator
        int max = (a > b) ? a : b;

        System.out.println("Maximum is: " + max);
    }
}
Output:
Maximum is: 20

🔍 Example 2: Checking Even or Odd

public class EvenOdd {
    public static void main(String[] args) {
        int number = 5;

        String result = (number % 2 == 0) ? "Even" : "Odd";

        System.out.println("Number is: " + result);
    }
}
Output:
Number is: Odd

💡 Tip: Nested Ternary Operator

You can nest ternary operators to check more than two conditions. 

int a = 10, b = 20, c = 15;
int max = (a > b) ? ((a > c) ? a : c) : ((b > c) ? b : c);
System.out.println("Maximum is: " + max);
Output: Maximum is: 20

🚫 When Not to Use Ternary Operator

Avoid using ternary operators if it makes the code harder to read. In such cases, use traditional if-else.

✅ Use ternary for simple conditional assignments.
❌ Avoid overusing or nesting too deeply.
