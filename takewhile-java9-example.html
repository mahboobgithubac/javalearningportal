Java 9 Stream.takeWhile() Example

‚úÖ What is takeWhile()?

takeWhile() processes elements from the stream until the predicate returns false. Once it fails, the rest of the stream is ignored, even if later elements match.

üî∏ Syntax

Stream<T> takeWhile(Predicate<? super T> predicate)

üîπ Example 1: Basic Use

List<Integer> numbers = List.of(2, 4, 6, 8, 3, 10, 12);
List<Integer> result = numbers.stream()
    .takeWhile(n -> n % 2 == 0)
    .collect(Collectors.toList());

System.out.println(result); // Output: [2, 4, 6, 8]
Note: It stops at the first element (3) which does not satisfy the predicate.

üîπ Example 2: With Strings

List<String> names = List.of("Apple", "Ant", "Axe", "Ball", "Bat");
List<String> startsWithA = names.stream()
    .takeWhile(name -> name.startsWith("A"))
    .collect(Collectors.toList());

System.out.println(startsWithA); // Output: [Apple, Ant, Axe]

üî∏ Difference: takeWhile() vs filter()

üîπ Example 3: Comparison

List<Integer> nums = List.of(2, 4, 6, 1, 8, 10);

List<Integer> usingTakeWhile = nums.stream()
    .takeWhile(n -> n % 2 == 0)
    .collect(Collectors.toList()); // [2, 4, 6]

List<Integer> usingFilter = nums.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList()); // [2, 4, 6, 8, 10]
