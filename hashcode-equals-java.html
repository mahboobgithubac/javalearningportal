Used to compare the content/state of two objects. Override it to compare fields instead of memory reference.

ðŸ”¹ hashCode() Method

Returns an integer that represents the hash value of the object. It's used in collections like HashMap or HashSet.

âœ… Example:

public class Student {
    private int id;
    private String name;

    public Student(int id, String name) {
        this.id = id;
        this.name = name;
    }

    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;

        Student student = (Student) obj;
        return id == student.id && name.equals(student.name);
    }

    @Override
    public int hashCode() {
        return 31 * id + name.hashCode();
    }

    public static void main(String[] args) {
        Student s1 = new Student(1, "Alice");
        Student s2 = new Student(1, "Alice");

        System.out.println(s1.equals(s2));        // true
        System.out.println(s1.hashCode() == s2.hashCode()); // true
    }
}

âœ… Rules to Remember:

Rule Description
If equals() returns true hashCode() must return same value
If hashCode() is same equals() may return true or false
Best practice Override both equals() and hashCode()
