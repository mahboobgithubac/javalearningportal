Summing Numbers Using Java 8 Stream API

Java 8 introduced the powerful Stream API, which allows you to perform functional-style operations on collections and arrays. One of the most common use-cases is calculating the sum of numbers.

‚úîÔ∏è Example 1: Sum of Integer List

List<Integer> numbers = Arrays.asList(10, 20, 30, 40);

int sum = numbers.stream()
                 .mapToInt(Integer::intValue)
                 .sum();

System.out.println("Sum = " + sum);
Explanation

- stream(): Converts the list into a Stream.
- mapToInt(): Converts each Integer to an int.
- sum(): Terminal operation to get the total.

‚úîÔ∏è Example 2: Sum Using reduce()

List<Integer> numbers = Arrays.asList(5, 15, 25, 35);

int sum = numbers.stream()
                 .reduce(0, Integer::sum);

System.out.println("Sum = " + sum);
Explanation

- reduce(0, Integer::sum) starts with 0 and adds each element.
- Useful for more flexible or custom logic.

‚úîÔ∏è Example 3: Sum of Object Field Values

class Product {
    int price;
    Product(int price) { this.price = price; }
    public int getPrice() { return price; }
}

List<Product> products = Arrays.asList(
    new Product(100),
    new Product(200),
    new Product(300)
);

int total = products.stream()
                    .mapToInt(Product::getPrice)
                    .sum();

System.out.println("Total Price = " + total);
Explanation

- This pattern is great for summing values from object fields.
- mapToInt() converts field values to a stream of int.

‚úîÔ∏è Example 4: Sum of Array Elements

int[] numbers = {7, 14, 21, 28};

int sum = Arrays.stream(numbers).sum();

System.out.println("Sum = " + sum);
Explanation

- Arrays.stream() turns array into IntStream.
- sum() then performs the accumulation.

‚úîÔ∏è Example 5: Sum Using IntStream.range()

int sum = IntStream.range(1, 6)  // 1 to 5
                   .sum();

System.out.println("Sum = " + sum);
Explanation

- IntStream.range(1, 6) creates a range from 1 to 5.
- Efficient for summing sequences without needing a collection.

üìù Summary

  • List + mapToInt + sum(): Clean and simple for summing `List<Integer>`.
  • List + reduce(): Offers control with a custom accumulator.
  • List of Objects + mapToInt(): Best for summing object fields like price or salary.
  • Array + Arrays.stream(): Most direct way to sum primitive arrays.
  • IntStream.range(): Ideal for summing over a numeric range (e.g., 1 to N).
