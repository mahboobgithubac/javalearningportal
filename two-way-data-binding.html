üîÅ Two-Way Data Binding in Angular

Two-way data binding in Angular allows synchronization of data between the component and the view (HTML). When the model changes, the view updates, and when the view changes, the model updates.

‚úÖ Key Concept

  • Property Binding: {{ value }} (Component ‚Üí View)
  • Event Binding: (input)="onInputChange($event)" (View ‚Üí Component)

Angular combines both into [(ngModel)].

üß† Syntax

<input [(ngModel)]="propertyName" />

üìò Example

1. Component (TypeScript)

// app.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent {
  username: string = 'Mahboob';
}

2. Template (HTML)

<!-- app.component.html -->
<h2>Welcome, {{ username }}</h2>
<input [(ngModel)]="username" placeholder="Enter your name" />

üéØ How It Works

  • Displays Welcome, Mahboob initially.
  • User updates the input field ‚Üí username in component updates.
  • {{ username }} in template reflects the updated value in real time.

‚úÖ Required Module

To use ngModel, import FormsModule:

// app.module.ts
import { NgModule } from '@angular/core';
import { FormsModule } from '@angular/forms';
import { BrowserModule } from '@angular/platform-browser';
import { AppComponent } from './app.component';

@NgModule({
  declarations: [AppComponent],
  imports: [BrowserModule, FormsModule],
  bootstrap: [AppComponent]
})
export class AppModule {}
