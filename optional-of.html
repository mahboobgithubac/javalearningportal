Java 8 Optional.of() Example

‚úÖ What is Optional.of()?

Optional.of() is a static factory method in Java used to create an Optional instance containing a non-null value.

üîπ Syntax:

Optional<T> optional = Optional.of(value);
  • value: A non-null object to wrap.
  • If the value is null, it throws a NullPointerException.
Note: Use Optional.of() only when you are sure the value is not null. Otherwise, use Optional.ofNullable().

‚úÖ Working Example:

import java.util.Optional;

public class OptionalOfExample {
    public static void main(String[] args) {
        String name = "Alice";
        Optional<String> optionalName = Optional.of(name);
        System.out.println("Optional contains: " + optionalName.get()); // Alice
    }
}

‚ùå Example That Throws Exception:

Optional<String> optionalNull = Optional.of(null); // Throws NullPointerException!
Error: Do not use Optional.of() with null values!

‚úÖ Safer Alternative:

Optional<String> optional = Optional.ofNullable(null); // Safe: Returns Optional.empty()
Use Optional.ofNullable() when the value might be null
