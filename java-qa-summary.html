Java Questions and Answers Summary

1. What is the output of try-catch block with 0/1?

try {
    return 0 / 1;
} catch (Exception ex) {
    return 2;
}

Answer: 0 — No exception is thrown.

2. What is the output of try-catch-finally with 1/0?

try {
    return 1 / 0;
} catch (Exception ex) {
    return 2;
} finally {
    return 3;
}

Answer: 3 — Finally block overrides return in catch.

3. Integer caching comparison using ==

Integer a = 100;
Integer b = 100;
Integer c = 150;
Integer d = 150;

System.out.println(a == b); // true
System.out.println(c == d); // false

Answer: Values from -128 to 127 are cached, so a == b is true, but c == d is false.

4. Stream map operation (square of list elements)

List<Integer> list = Arrays.asList(2, 4, 5, 8);
List<Integer> result = list.stream().map(x -> x * x).collect(Collectors.toList());

Output: [4, 16, 25, 64]

5. Flatten list of lists using flatMap

List<String> namesList1 = Arrays.asList("John", "Jane");
        List<String> namesList2 = Arrays.asList("Tom", "Jerry");
        List<String> namesList3 = Arrays.asList("Alice", "Bob");

        List<List<String>> namesNested = Arrays.asList(namesList1, namesList2, namesList3);

        // Flattening the nested list using flatMap
        List<String> flatList = namesNested.stream()
                                            .flatMap(List::stream)
                                            .collect(Collectors.toList());

        System.out.println(flatList);

Output: [Alice, Bob, John, Johnny, Anu, Ammay]

6. Functional Interface and Lambda Example

@FunctionalInterface
interface Calculator {
    int calculate(int x, int y);
}

Calculator add = (x, y) -> x + y;
Calculator subtract = (x, y) -> x - y;

System.out.println(add.calculate(5, 3)); // 8
System.out.println(subtract.calculate(5, 3)); // 2

7. Convert Lambda to Method Reference

public static int addOp(int x, int y) { return x + y; }
public static int subtractOp(int x, int y) { return x - y; }

Calculator add = ClassName::addOp;
Calculator subtract = ClassName::subtractOp;

8. Varargs Method with Dynamic Arguments

public static int doCalculation(int... args) {
    int sum = 0;
    for (int num : args) {
        sum += num;
    }
    return sum;
}

int result = doCalculation(2, 3, 5, 10); // 20

9. What is @PropertySource in Spring?

@Configuration
@PropertySource("classpath:application.properties")
public class AppConfig {

    @Value("${app.name}")
    private String appName;

    @Value("${app.version}")
    private String appVersion;
}

Purpose: Loads external properties into Spring environment for injection with @Value.
