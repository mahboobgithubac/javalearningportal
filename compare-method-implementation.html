compare() Implementation in Java — Quick Reference

Syntax

int compare(T o1, T o2)

Return values:

  • 0 if o1 equals o2
  • < 0 if o1 should come before o2
  • > 0 if o1 should come after o2

Example: Custom Comparator to Sort by Age

import java.util.*;

class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() { return name; }
    public int getAge() { return age; }
}

class AgeComparator implements Comparator {
    @Override
    public int compare(Person p1, Person p2) {
        return Integer.compare(p1.getAge(), p2.getAge());
        // OR: return p1.getAge() - p2.getAge();
    }
}

// Usage example below

Usage Example

public class Main {
    public static void main(String[] args) {
        List list = new ArrayList<>();
        list.add(new Person("Alice", 30));
        list.add(new Person("Bob", 25));
        list.add(new Person("Charlie", 35));

        Collections.sort(list, new AgeComparator());

        for (Person p : list) {
            System.out.println(p.getName() + " - " + p.getAge());
        }
    }
}

To Compare by Name

class NameComparator implements Comparator {
    public int compare(Person p1, Person p2) {
        return p1.getName().compareTo(p2.getName());
    }
}

compareTo() vs compare()

Quick feature comparison.

Feature Interface Defined In Used For Modifies Class?
compareTo() Comparable<T> The class itself Natural ordering Yes
compare() Comparator<T> External class or lambda Custom ordering No
Tip: Use lambda expressions for compact comparators, e.g. Comparator.comparing(Person::getAge).
