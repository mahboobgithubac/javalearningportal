RestTemplate Methods in Spring Boot

1. getForObject()

Sends a GET request and returns the response body as an object.

String result = restTemplate.getForObject("http://example.com/api/data", String.class);

2. getForEntity()

Sends a GET request and returns the entire ResponseEntity (status, headers, and body).

ResponseEntity<String> response = restTemplate.getForEntity("http://example.com/api/data", String.class);

3. postForObject()

Sends a POST request and returns the response body as an object.

MyResponse res = restTemplate.postForObject(
    "http://example.com/api/save",
    requestObject,
    MyResponse.class);

4. postForEntity()

Sends a POST request and returns the entire ResponseEntity.

ResponseEntity<MyResponse> response = restTemplate.postForEntity(
    "http://example.com/api/save",
    requestObject,
    MyResponse.class);

5. put()

Sends a PUT request; returns nothing.

restTemplate.put("http://example.com/api/update/1", updatedObject);

6. delete()

Sends a DELETE request.

restTemplate.delete("http://example.com/api/delete/1");

7. exchange()

Flexible method allowing custom HTTP methods and headers.

HttpHeaders headers = new HttpHeaders();
headers.set("Authorization", "Bearer token");
HttpEntity<String> entity = new HttpEntity<>(headers);

ResponseEntity<String> response = restTemplate.exchange(
    "http://example.com/api/secure-data",
    HttpMethod.GET,
    entity,
    String.class);

8. execute()

Lower-level method with full control over request and response handling.

restTemplate.execute(
    "http://example.com/api/custom",
    HttpMethod.GET,
    null,
    clientHttpResponse -> {
        return new BufferedReader(
            new InputStreamReader(clientHttpResponse.getBody())).readLine();
    });
