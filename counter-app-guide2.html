Angular Counter App – Q&A Guide

Setup Component Template Style App Component Run

Q1: How do I set up a new Angular application for a counter app?

Run the following commands in your terminal:

ng new counter-app
cd counter-app
ng serve

Q2: How do I create a counter component and implement the logic?

Use Angular CLI to generate a new component:

ng generate component counter

Then add the following logic to counter.component.ts:

import { Component } from '@angular/core';

@Component({
  selector: 'app-counter',
  templateUrl: './counter.component.html',
  styleUrls: ['./counter.component.css']
})
export class CounterComponent {
  count: number = 0;

  increment(): void {
    this.count++;
  }

  decrement(): void {
    if (this.count > 0) {
      this.count--;
    }
  }

  reset(): void {
    this.count = 0;
  }
}

Q3: What is the HTML structure for the counter component?

Add this to counter.component.html:

<div style="text-align: center; margin-top: 50px;">
  <h2>Angular Counter App</h2>
  <h1>{{ count }}</h1>

  <button (click)="increment()">Increment</button>
  <button (click)="decrement()">Decrement</button>
  <button (click)="reset()">Reset</button>
</div>

Q4: How should I style the counter buttons?

Add the following CSS to counter.component.css:

button {
  margin: 0 10px;
  padding: 10px 20px;
  font-size: 18px;
  cursor: pointer;
}

Q5: How do I use the counter component in the main application?

Update app.component.html like this:

<app-counter></app-counter>

Q6: How do I run and view the Angular counter app?

Run the Angular development server:

ng serve

Then open your browser and navigate to: http://localhost:4200
