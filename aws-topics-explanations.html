AWS EC2 (Elastic Compute Cloud)

Provides scalable virtual servers in the cloud. Used to run applications on virtual machines (instances).

AWS S3 (Simple Storage Service)

Highly scalable object storage service to store and retrieve any amount of data at any time.

AWS Lambda

Run code without provisioning or managing servers. It executes code in response to triggers like S3 uploads or API Gateway events.

AWS Elastic Beanstalk

PaaS(Platform as a Service) that automates deployment and scaling of applications using services like EC2, S3, and RDS.

AWS SQS (Simple Queue Service)

Managed message queuing service for decoupling and scaling microservices and distributed systems.

AWS SNS (Simple Notification Service)

Managed publish/subscribe messaging service to send notifications to email, SMS, or other endpoints.

AWS ECR (Elastic Container Registry)

Fully managed Docker container registry for storing, sharing, and deploying container images.

AWS ECS (Elastic Container Service)

Managed container orchestration service that supports Docker containers and integrates with ECR.

AWS CodePipeline

CI/CD service for automating build, test, and deploy phases of application release pipelines.

AWS RDS (Relational Database Service)

Managed relational database service that supports databases like MySQL, PostgreSQL, and Oracle.

AWS CloudWatch

Monitoring and observability service for AWS resources and applications, including metrics, logs, and alarms.
