Find Duplicates in an Integer Array Using Java Stream API

Java 8 introduced the powerful Stream API, which allows for functional-style operations on collections and arrays. One common use case is identifying duplicate elements in an array. Here's how to do it in a clean, concise way using Streams.

Step-by-Step Approach

  1. Convert the array to a stream.
  2. Track seen elements using a Set.
  3. Use filter() to identify duplicates.
  4. Collect duplicates using Collectors.toSet().

Java Code Example

import java.util.*;
import java.util.stream.*;

public class DuplicateFinder {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 2, 4, 5, 1, 6, 3};

        Set<Integer> duplicates = findDuplicates(numbers);
        System.out.println("Duplicates: " + duplicates);
    }

    public static Set<Integer> findDuplicates(int[] arr) {
        Set<Integer> seen = new HashSet<>();

        return Arrays.stream(arr)
                .filter(num -> !seen.add(num)) // if add returns false, it's a duplicate
                .boxed()
                .collect(Collectors.toSet());
    }
}

Output

Duplicates: [1, 2, 3]
Note: The use of boxed() is necessary to convert an IntStream to a Stream<Integer> before collecting.

Why This Works

The key idea here is using the return value of Set.add(). If the element is already in the set, add() returns false, and thatâ€™s how we detect duplicates using filter().

Conclusion

Using the Stream API, we can efficiently detect duplicates in a functional style without writing explicit loops. This is clean, concise, and ideal for modern Java development.
