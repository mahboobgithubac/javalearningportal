Top Java 8 Interview Questions and Answers

This blog covers frequently asked Java 8 interview questions with answers and code examples. Great for job seekers and Java enthusiasts!

1. What are the main features introduced in Java 8?

Answer: Key features include:

  • Lambda Expressions
  • Stream API
  • Functional Interfaces
  • Default and Static Methods in Interfaces
  • Optional Class
  • Date and Time API
  • Method and Constructor References

2. What is a Lambda Expression?

Answer: A lambda expression is an anonymous function. It enables functional programming.

List<String> list = Arrays.asList("A", "B", "C");
list.forEach(s -> System.out.println(s));

3. What is a Functional Interface?

Answer: An interface with a single abstract method. Annotated with @FunctionalInterface.

@FunctionalInterface
interface MyFunc {
    void execute();
}

4. What is the Stream API?

Answer: The Stream API provides functional-style operations on collections.

List<Integer> list = Arrays.asList(1, 2, 3, 4, 5);
int sum = list.stream()
              .filter(n -> n % 2 == 0)
              .mapToInt(Integer::intValue)
              .sum();

5. What is the difference between map() and flatMap()?

Answer:

  • map(): Transforms each element; may return nested streams.
  • flatMap(): Flattens multiple streams into one.
List<String> words = Arrays.asList("Hello", "World");

List<String> chars = words.stream()
    .flatMap(w -> Arrays.stream(w.split("")))
    .collect(Collectors.toList());

6. What is the Optional class?

Answer: Optional is a container to avoid null values.

Optional<String> name = Optional.ofNullable(getName());
name.ifPresent(System.out::println);

7. How are default methods used in interfaces?

Answer: Java 8 allows default methods with implementation in interfaces.

interface MyInterface {
    default void show() {
        System.out.println("Default Method");
    }
}

8. What is the new Date and Time API?

Answer: Java 8 introduced java.time package for better date/time handling.

LocalDate today = LocalDate.now();
LocalDate birthday = LocalDate.of(1990, Month.JANUARY, 1);

9. How do you use Method References?

Answer: Method references shorten lambda syntax.

list.forEach(System.out::println);

10. What are some built-in functional interfaces?

Answer: Java 8 provides many functional interfaces in java.util.function:

  • Predicate<T> – returns boolean
  • Function<T,R> – takes T, returns R
  • Supplier<T> – supplies a value
  • Consumer<T> – performs an action

✅ Summary

Java 8 brought functional programming into Java. Understanding lambda, streams, and Optionals is key for modern Java development and interviews.
