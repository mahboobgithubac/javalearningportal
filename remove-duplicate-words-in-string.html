Java 8: Remove Duplicate Words

Question 1: How do you remove duplicate words from a string using basic Java?

Show Answer

Example:

String input = "Java is is a programming programming language";

Expected Output: Java is a programming language

Java Code:

import java.util.LinkedHashSet;
import java.util.Set;

public class RemoveDuplicateWords {
    public static void main(String[] args) {
        String input = "Java is is a programming programming language";
        String result = removeDuplicateWords(input);
        System.out.println(result);  // Output: Java is a programming language
    }

    public static String removeDuplicateWords(String input) {
        String[] words = input.split("\\s+"); // split by spaces
        Set<String> uniqueWords = new LinkedHashSet<>();

        for (String word : words) {
            uniqueWords.add(word);
        }

        return String.join(" ", uniqueWords);
    }
}

Question 2: How do you remove duplicate words from a string using Java 8 Stream API?

Show Answer

Java 8 Code:

import java.util.Arrays;
import java.util.stream.Collectors;

public class RemoveDuplicateWordsJava8 {
    public static void main(String[] args) {
        String input = "Java is is a programming programming language";
        String result = Arrays.stream(input.split("\\s+"))
                              .distinct()
                              .collect(Collectors.joining(" "));
        System.out.println(result);  // Output: Java is a programming language
    }
}

Explanation:

  • split("\\s+"): Splits the string into words by spaces.
  • Arrays.stream(...): Creates a stream from the array.
  • .distinct(): Removes duplicates while preserving order.
  • Collectors.joining(" "): Joins the words with a space.
