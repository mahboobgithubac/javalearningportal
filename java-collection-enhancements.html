Java Collection Enhancements (Java 9 to 17)

1. Factory Methods for Collections (Java 9)

List names = List.of("Alice", "Bob", "Charlie");
Set nums = Set.of(1, 2, 3);
Map map = Map.of(1, "One", 2, "Two");

Immutable collections that throw exceptions on modification and do not allow nulls.

2. Convenient Map.entry() for Map.ofEntries()

Map map = Map.ofEntries(
    Map.entry(1, "One"),
    Map.entry(2, "Two"),
    Map.entry(3, "Three")
);

3. Stream API Enhancements (Java 9)

List result = List.of(1, 2, 3, 4, 5, 0, 6, 7).stream()
    .takeWhile(n -> n > 0)
    .collect(Collectors.toList());

4. Collectors Enhancements (Java 9)

Map> partitioned = List.of(1, 2, 3, 4, 5).stream()
    .collect(Collectors.partitioningBy(n -> n % 2 == 0,
        Collectors.filtering(n -> n > 2, Collectors.toList())));

5. Immutable Collections via List.copyOf(), Set.copyOf() (Java 10)

List copyList = List.copyOf(originalList);
Set copySet = Set.copyOf(originalSet);

6. Optional.toStream() (Java 9)

Optional name = Optional.of("Java");
name.stream()
    .map(String::toUpperCase)
    .forEach(System.out::println);

7. Stream.ofNullable() (Java 9)

Stream stream = Stream.ofNullable(null); // Empty stream

8. Performance Improvements (Java 11 - 17)

Internal enhancements for memory usage, class data sharing, and iteration performance for core collections.

9. Collectors.teeing() (Java 12)

var result = Stream.of(1, 2, 3, 4, 5)
    .collect(Collectors.teeing(
        Collectors.summingInt(i -> i),
        Collectors.counting(),
        (sum, count) -> sum / count
    ));

10. Pattern Matching with Collections (Java 16)

if (obj instanceof List<String> list) {
    list.forEach(System.out::println);
}
