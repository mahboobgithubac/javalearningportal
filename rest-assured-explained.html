Rest Assured is a Java-based library used for testing RESTful APIs. It provides a domain-specific language (DSL) for writing powerful, readable, and maintainable tests for REST services.

ðŸ‘‰ Think of Rest Assured as a tool for testing REST APIs, not the same thing as a REST API.

ðŸ”¸ Key Differences:

Term Meaning
REST API An API that follows REST (Representational State Transfer) principles; it's what you're testing.
Rest Assured A Java library/tool used to test REST APIs by sending HTTP requests and validating the responses.

ðŸ”¹ Why use Rest Assured?

  • Easy syntax for sending HTTP requests
  • Built-in support for JSON/XML path parsing
  • Integrated with JUnit/TestNG
  • Validations like status codes, headers, body content, etc.

ðŸ”¸ Example of Using Rest Assured in Java:

import io.restassured.RestAssured;
import static io.restassured.RestAssured.*;
import static org.hamcrest.Matchers.*;

public class ApiTest {
    public static void main(String[] args) {
        RestAssured.baseURI = "https://jsonplaceholder.typicode.com";

        given()
            .when()
                .get("/posts/1")
            .then()
                .statusCode(200)
                .body("id", equalTo(1))
                .body("userId", notNullValue());
    }
}
