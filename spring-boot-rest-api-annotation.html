Spring Boot and REST API Annotations Cheatsheet

✅ Core Spring Boot Annotations

AnnotationDescription
@SpringBootApplicationCombines @Configuration, @EnableAutoConfiguration, and @ComponentScan. Entry point of a Spring Boot app.
@EnableAutoConfigurationEnables Spring Boot’s auto-configuration mechanism.
@ComponentScanTells Spring where to search for annotated components.
@ConfigurationIndicates that the class can be used by Spring IoC as a source of bean definitions.

✅ Spring Bean & Dependency Injection Annotations

AnnotationDescription
@ComponentGeneric stereotype for Spring-managed components.
@ServiceSpecialization of @Component for service layer classes.
@RepositorySpecialization of @Component for DAO classes. Adds exception translation.
@AutowiredAutomatically injects dependencies by type.
@QualifierSpecifies the exact bean to inject when multiple are present.
@PrimaryMarks a bean as the primary choice when multiple candidates are present.
@LazyMarks bean to be lazily initialized.
@Value("${property}")Injects value from application.properties.

✅ REST API (Spring Web MVC) Annotations

AnnotationDescription
@RestControllerCombines @Controller and @ResponseBody for REST APIs.
@RequestMappingMaps HTTP requests to handler methods/classes.
@GetMapping, @PostMapping, @PutMapping, @DeleteMapping, @PatchMappingShortcut for @RequestMapping(method = ...) for respective HTTP methods.
@PathVariableBinds method parameter to a URI template variable.
@RequestParamExtracts query parameters from URL.
@RequestBodyBinds the request body to a method parameter (usually JSON to object).
@ResponseBodyReturns object data (like JSON/XML) in the response body.
@ResponseStatusMarks method/exception with a specific HTTP status.
@CrossOriginEnables Cross-Origin Resource Sharing (CORS) support.

✅ Data Persistence (Spring Data JPA)

AnnotationDescription
@EntityMarks a class as a JPA entity (table).
@IdSpecifies the primary key of an entity.
@GeneratedValueStrategy to auto-generate primary key values.
@Table(name = "table_name")Maps entity to a specific table.
@Column(name = "column_name")Maps a field to a specific column.
@OneToOne, @OneToMany, @ManyToOne, @ManyToManyDefines JPA relationships between entities.
@JoinColumnSpecifies foreign key column for relationships.

✅ Validation (Spring Boot + Hibernate Validator)

AnnotationDescription
@Valid / @ValidatedTriggers bean validation on a method parameter or field.
@NotNull, @NotBlank, @Size, @Min, @Max, @EmailBean validation constraints for fields.
@Pattern(regexp = "...")Validates string values using a regex pattern.

✅ Spring Boot Testing Annotations

AnnotationDescription
@SpringBootTestLoads full Spring context for integration testing.
@WebMvcTestFor testing Spring MVC controllers.
@DataJpaTestFor testing JPA repositories.
@MockBeanCreates and injects a mock bean.
@TestConfigurationCustom configuration for test context.

✅ Other Useful Annotations

AnnotationDescription
@ScheduledUsed to schedule tasks at fixed intervals.
@EnableSchedulingEnables scheduled tasks.
@AsyncMarks a method to run asynchronously.
@EnableAsyncEnables asynchronous method execution.
@ProfileSpecifies that a bean is loaded only in specific Spring profiles.
@PropertySourceLoads properties from an external .properties file.
@BeanDeclares a bean inside a @Configuration class.
@DependsOnDefines the order of bean initialization.
